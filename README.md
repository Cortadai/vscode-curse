Este es un **repositorio educativo de curso de Visual Studio Code** construido con Angular 7, diseñado para enseñar trucos, atajos y técnicas de edición eficiente en VSCode. El proyecto está organizado en 5 módulos temáticos que cubren: edición básica (movimientos, comentarios, terminal, zen mode, emmet), edición multi-cursor (clonar líneas, capitalizar, múltiples ocurrencias), navegación de clases y definiciones (ir a línea, markdown, reemplazo), snippets personalizados, y extensiones útiles (Bookmarks, Bracket Pair Colorizer, Color Highlight, Live Server, Material Theme, Paste JSON as Code, y TODO Highlight). Cada módulo contiene archivos de ejemplo en HTML, TypeScript, CSS y otros formatos que sirven como ejercicios prácticos para aprender las funcionalidades de VSCode de manera interactiva.
